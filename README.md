
# NLP Assignment Projects

## Overview
This repository contains three assignments completed for the Natural Language Processing course at University of Naples "Parthenope". Each assignment explores different fundamental aspects of NLP, from basic text preprocessing to advanced language modeling techniques.

Key Topics:
